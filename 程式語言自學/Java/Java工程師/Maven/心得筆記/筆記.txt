[Maven筆記]

(一)起源
    由Apache開源的項目
(二)功能
    一種Java統一的管理方式，包含工程的構建和管理，
    例如:第三方庫的下載和引用、應用程式打包(.jar或.war)...等等
(三)特性
    1.設置工程架構的規則統一:例如，哪個資料夾存放資源，哪個資料夾存放程式碼...等等。
    2.強大的第三方依賴管理:例如，對依賴的組件或套件進行自動下載與更新。
    3.可擴展插件機制:例如，覺得Maven哪裡無法滿足我們的業務，能自己寫插件或使用別人寫的插件注入Maven之中。
(四)加入Dependency方法
    先到以下網址搜尋要安裝的套件
    https://search.maven.org/
    然後複製依賴字串到pom.xml中，參考:Maven加入依賴.png
(五)打包方法
    參考pom.xml檔案
